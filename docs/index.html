Project Theme Interests

General Theme

Using environmental data to look at long term changes in places I am familiar with.

What I Am Interested In

I am interested in working with real environmental datasets and learning how to explore trends over time at a local scale. Starting with one city feels manageable and makes it easier to understand what the data is actually showing.

Possible Direction

One possible direction for this project is looking at historical temperature data for Cedar Rapids, Iowa to see how average temperatures may have changed over time. This is just an initial idea and not a final project decision.

Tools and Skills I Want to Work With

  • Public environmental and climate datasets
  • Basic data analysis and plotting
  • Geospatial or cloud based tools like Google Earth Engine
  • Git and GitHub for organizing files and tracking changes

Status

This page describes a preliminary theme of interest. The exact topic, data, and methods will likely change once I spend more time exploring what is available and what is realistic to do this semester.
